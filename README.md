# Python-Term-Project-Movie-Ticket-Booking-System

Özellikler:
Seansları listeleme,
Koltuk durumlarını görme (Boş / Dolu),
Koltuk rezervasyonu yapma,
Yapılan rezervasyonları listeleme,
Rezervasyon iptal etme

Mevcut Seanslar:
Batman – 20:00,
Space Odyssey – 18:30
Her seans için 10 koltuk bulunmaktadır.

Kullanılanlar:
list, dict, tuple, if/else, for, while, function, input, print
